### Architecture

- Safety scoring: weighted formula across 5 civic dimensions (lighting, accidents, flood, surface, walkability)
- Data sources: OpenStreetMap (Overpass API), HYDRAA flood zones, MoRTH accident data, citizen reports
- AI: Claude Vision API (primary) with HuggingFace + CLIP fallback
- No crime data, no PII stored — anonymous reporting only
